# MedGuardian AI 💊🤖

An advanced all-in-one medicine reminder and health management application built with Python.  MedGuardian AI helps individuals and families manage medications, track adherence, receive smart reminders, predict refills, monitor health journals, and identify potential drug interactions through a modern desktop experience.

---

## ✨ Features

### 🔐 User Management
- User registration and login
- Secure password hashing
- Multiple user profiles
- Family member management

### 💊 Medicine Management
- Add, edit, and delete medicines
- Custom dosage schedules
- Daily, weekly, and interval reminders
- Medicine categories and notes
- Before/after meal instructions

### 🔔 Smart Reminder System
- Desktop notifications
- Sound alerts
- Snooze reminders
- Missed-dose detection
- Automatic adherence tracking

### 📊 Analytics Dashboard
- Daily adherence percentage
- Weekly and monthly reports
- Most missed medications
- Medicine usage statistics
- Interactive charts and graphs

### 📦 Refill Prediction
- Pill quantity tracking
- Remaining dose estimation
- Refill date prediction
- Low-stock warnings

### 👨‍👩‍👧‍👦 Family Health Management
- Multiple patient profiles
- Parent and child medicine schedules
- Shared dashboards
- Family-wide reports

### 📖 Health Journal
- Daily health notes
- Symptom tracking
- Side-effect logging
- Recovery progress monitoring

### ⚠️ Drug Interaction Checker
- Detect common medicine interactions
- Warning notifications
- Safety recommendations
- Educational information

### 🎨 Modern Desktop Interface
- Clean GUI built with Tkinter
- Dark and light themes
- Reusable UI components
- User-friendly navigation
