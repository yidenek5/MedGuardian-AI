# MedGuardian AI 💊🤖

An intelligent medicine management platform built with Python.

MedGuardian AI helps individuals and families manage medications through smart reminders, adherence tracking, refill prediction, health journals, and drug interaction warnings.

## ✨ Features

- 🔐 User authentication
- 💊 Medicine management
- 🔔 Smart desktop reminders
- 📊 Adherence analytics
- 📦 Refill prediction
- 👨‍👩‍👧 Family profiles
- 📖 Health journals
- ⚠️ Drug interaction alerts
- 🎨 Modern Tkinter GUI

## 🛠️ Tech Stack

- Python 3
- Tkinter
- SQLite
- Plyer
- Matplotlib
- APScheduler
- Pytest
- Git & GitHub
